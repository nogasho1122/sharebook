# DEMO
 
ユーザーが自分の読んだ本について画像と文字で自由に投稿しシェアできるアプリです。  
コメント機能があるので他ユーザにーにコメントすることができます。
 
# Features
 
ログイン、ログアウト機能がついているのでアプリに登録して楽しんでいただくことができます。  
新規登録時にニックネームすの登録ができるのでニックネームをつけることもできます。
 
# Requirement
 
* ruby 2.5.1
